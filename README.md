# Poke-Showcase-Dex
Pokemon Go players, figure out your showcase score using Poke Showcase Dex! Manage your storage and know which ones to keep or turn to candy, without having an active showcase.
* Easily scan Pokemon with the overlay or manually enter the details
* Get an estimate of your score or enter your score if you know it
* See your highest scored mon by type or species
* Generate a search string to find the pokemon
* Import from Pokegenie
